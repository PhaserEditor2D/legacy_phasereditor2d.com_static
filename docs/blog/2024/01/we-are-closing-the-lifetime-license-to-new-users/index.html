We are closing the Lifetime License to new users

January 07, 2024 - Posted in news

Dear,

As I announced in my previous post, we are integrating Phaser Editor 2D into the new Phaser Studio services.

Merging into a company, in addition to funding the Phaser Editor 2D development, introduces new business models and services. For that reason, I’m closing the Phaser Editor 2D Lifetime License. This means new users of Phaser Editor 2D can opt only for the Phaser Editor 2D Backers licenses.

Owners of the Phaser Editor 2D Backers licenses pay every month, but this license may be stopped at any moment in the next months. Sure, you will get access to the editor for the period you purchased.

What about the current owners of the Phaser Editor 2D Lifetime License?

If you own a Phaser Editor 2D Lifetime License you can continue using this license key for unlocking the editor. As soon as Phaser Editor 2D is integrated into Phaser Studio, we will provide you access to the desktop edition of the editor.

If you have any doubt, please, contact us at developers@phasereditor2d.com or in our Discord server.

Keep in touch!

Arian
