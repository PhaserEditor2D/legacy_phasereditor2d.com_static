Welcome the uPhaserHelpCenter desktop app!

November 30, 2022 - Posted in release

Hi!

Let’s welcome a new product: the uPhaserHelpCenter desktop app!

It is possible that you knew about the Unofficial Phaser Help Center website. It is a web application for browsing the Phaser docs, examples, and code. It had been around for a while. However, now you have a desktop version of the same app.

This desktop version has the advantage that you can run it offline, and if you have a no so speedy Internet, it can improve how you access the Phaser docs.

I’m hosting the uPhaserHelpCenter as a Gumroad product, but it is free. You have to pay $0. You only have to provide your email, and Gumroad will send you the links to the installation files.

I invite you to watch this intro video. I’m pretty sure with this tool you will save a lot of time on developing Phaser games.

Keep in contact!
