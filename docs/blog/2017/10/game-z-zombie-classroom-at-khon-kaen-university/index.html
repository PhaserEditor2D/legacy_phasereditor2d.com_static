Game: Z-Zombie (classroom at Khon Kaen University)

October 16, 2017 - Posted in made-with-pe

ZZombie is a side-scrolling action platform game that is made with Phaser.
This game was created for education purpose in Basic of Game Programming classroom at Khon Kaen University.

  • GitHub repository at https://github.com/twachikku/zzombie_phaser
  • Play on Web Browser
  • Play on Android
  • Play on Chrome OS
  • Play on iOS at … coming soon …

Game Play

  • To Walk, press the arrow keys left or right.
  • To Jump, press the arrow key up.
  • To Attack, press the key “X”.

Tools

The list of tools are used in this project.

  • Phaser the core game SDK.
  • Phaser Editor A friendly IDE for HTML5 games programming.
  • Tiled A Tiled Map Editor
  • Spriter 2D sprite animation character creation software.
  • Audacity Audacity is free, open source, audio software for multi-track recording and editing.
