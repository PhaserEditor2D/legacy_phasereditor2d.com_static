Phaser Editor v1.5.3 released

November 08, 2018 - Posted in release

Hi,

Today we are releasing v1.5.3 of Phaser Editor. This is a minor release to fix bugs reported by the community:

#89: UTF8 text error in canvas file About to load Canvas files with UTF-8 characters.

#86: State “Level.canvas” showing error “Failed to create the part’s controls” on MacOS Sierra About to a missing font in macOS with Sierra and Retina display.

If you experience any error with the editor, please let us know, we will keep updating the v1 branch while users request it.

Best regards, Arian
