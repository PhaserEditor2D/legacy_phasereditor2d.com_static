Hi,

Today I'm releasing a minor version 3.31.2 with bug fixes.

[Download Phaser Editor 2D v3.31.2](https://phasereditor2d.com/downloads)

* Starter templates: webpack build split output in chunks (main & phaser).
* Fixes webpack build of "A day in the beach" example.
* Monaco editor loads Phaser built-in type definitions. It doesn't require reading the `.d.ts` files from the project.
* Inspector view: fixes the clicking on the three-dots menu of sections. Now it doesn't collapse the section.
* Installing Phaser Editor 2D via NPM now sets the exec permissions to `PhaserEditor2D` file (Linux & macOS).

(I keep working in the marketplace!)
