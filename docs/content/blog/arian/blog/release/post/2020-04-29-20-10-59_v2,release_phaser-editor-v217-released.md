## Phaser 3.23.0 {#phaser-3-23-0}

The latest version Phaser is now available in the editor. It is included in the project wizards, the Scene Editor, and the Phaser Labs tools. 

[Read the Phaser v3.23.0 release notes](https://phaser.io/news/2020/04/phaser-3230-released)

The Phaser runtime files of your old projects are not updated. If you wish to update your game runtime to the new Phaser version you should do it manually. [Learn more about it in the docs](/docs/v2/workbench.html#phaser-version-update).

## Phaser Editor v3 {#phaser-editor-v3}

We keep working on the web-based IDE for Phaser development. Right now, we are writing the documentation. It is a huge task (oh, my poor English skills!), but there are just a few chapters to complete. We plan to release the first general-available version very soon, at the end of May.

In addition to the v3 release, we should do a re-design of the Phaser Editor website, because right now we are maintaining three very different versions of the editor, and some users are not clear about that.
