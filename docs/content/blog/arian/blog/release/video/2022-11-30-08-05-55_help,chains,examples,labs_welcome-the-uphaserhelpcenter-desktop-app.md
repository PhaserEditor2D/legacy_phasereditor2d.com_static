Let's welcome a new product: the [uPhaserHelpCenter desktop app](https://fornaris.gumroad.com/l/uPhaserHelpCenter)!

It is possible that you knew about the [Unofficial Phaser Help Center website](http://helpcenter.phasereditor2d.com/). It is a web application for browsing the Phaser docs, examples, and code. It had been around for a while. However, now you have a desktop version of the same app.

This desktop version has the advantage that you can run it offline, and if you have a no so speedy Internet, it can improve how you access the Phaser docs.

I'm hosting the uPhaserHelpCenter as a [Gumroad product](https://fornaris.gumroad.com/l/uPhaserHelpCenter), but it is free. You have to pay $0. You only have to provide your email, and Gumroad will send you the links to the installation files.

I invite you to [watch this intro video](https://www.youtube.com/watch?v=kKvZ-mgtZfw&list=PLB8gI_5U0MvBA9ctHhShCmgOXRYwqPrbg&index=1&). I'm pretty sure with this tool you will save a lot of time on developing Phaser games.
