The user Wachirawut Thamviset created a video about to make a dinosaur animation in Spriter and Phaser Editor.

This tutorial shows in details how to create the dino parts in Photoshop, how to animate them in Spriter and how to import everything in Phaser Editor, to create an animated scene.

The author creates the scene by coding everything (he does not use the [scene maker animations][1]), and it is an example of how the JS editor helps with the Phaser API.

  [1]: https://phasereditor2d.com/docs/canvas.html#animations
