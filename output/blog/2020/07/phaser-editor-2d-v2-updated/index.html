Phaser Editor 2D v2 updated!

Hi!

We just updated Phaser Editor 2D v2!

If you are using this version of the editor (the one based on Eclipse), we recommend migrating to v3 (the one based on the browser).

We know many of you are working on projects based on v2, so we are updating it each time Phaser 3 releases a new version.

This means this Phaser Editor 2D v2.1.8 release includes Phaser v3.24.1.
