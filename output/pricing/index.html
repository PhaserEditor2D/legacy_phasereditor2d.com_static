Licenses & Pricing

What you need to know about Phaser Editor 2D licenses and pricing.

Phaser Editor 2D Lifetime License

Important: The Lifetime License is closed to new users. Read more.

Phaser Editor 2D, by default, runs in a Free mode. In this mode, all features are available, but projects are limited to a maximum number of 200 files.

To run the IDE in the Premium mode (without limits), you should purchase a license.

With a license key, you can unlock any Phaser Editor 2D version. Including any future version. You only have to take care of the expiration date of the license.

The licenses are personal. If you have any question, write us an email to the address: developers@phasereditor2d.com.

About

Phaser Editor 2D is a multi-platform IDE to develop games based on the fast and popular Phaser framework. It is a commercial product made and supported by Arian Fornaris.

Phaser is a free and open source software developed and owned by Richard Davey at www.photonstorm.com. You can visit their funding page and help them to make Phaser even better.
