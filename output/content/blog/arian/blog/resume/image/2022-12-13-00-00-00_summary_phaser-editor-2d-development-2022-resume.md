This is a small summary of what I did in 2022, in relation to Phaser Editor 2D:

## Phaser Editor 2D in 2022

### January

- Published the Phaser Editor 2D Marketplace.

### April 

- New user properties:
  - Scene Key property
 - Object Variable property
 - Color property
- New Game Object parameter
- Paste in place command
- Apple M1 support

### May

- New Object constructor user property
- New project templates:
  - CapacitorJS project template
  - Custom web font project template

### June

- Plugins updates
  - NinePatchImage
  - NinePatchContainer
- New plugin RoundedRectangleGraphics

### August

- New Polygon game object.
- New origin commands in the Layout toolbar

### October

- New Arcade physics support
- Prefab improvements: adding children to instances.

### November

- New video series Making your first Phaser game.
- User components navigation in the components editor and scene editor improvements.
- Prefab’s user properties navigation improvements.
- New animations editor compiler.
- New Keyboard.Key scene object.
- New uPhaserHelpCenter desktop app.

### December

- New pricing & free plan.
- New License Manager service.
