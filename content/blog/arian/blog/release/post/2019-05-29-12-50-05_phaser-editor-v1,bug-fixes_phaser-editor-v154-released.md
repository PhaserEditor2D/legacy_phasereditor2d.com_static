Hi!

Today we are releasing a new version of the v1 series of the editor: v1.5.4.

The v1 of the IDE is not in active development, we are releasing only bug fixes. Especially, we are fixing the bugs the clients are reporting.

In this version we fixed:

[#118: Spelling mistake on Atlas Generator Tooltip](https://github.com/PhaserEditor2D/PhaserEditor/issues/118)

[#112: Adding bitmap text does not work](https://github.com/PhaserEditor2D/PhaserEditor/issues/112)

[#124: BitmapText item doesn't display in the right place when re-opening canvas file](https://github.com/PhaserEditor2D/PhaserEditor/issues/124)

[#119: Generated typescript creates compile error when setting scale on bitmaptext](https://github.com/PhaserEditor2D/PhaserEditor/issues/119)

[#117: Optional parameters in constructor not generated as optional when extending class](https://github.com/PhaserEditor2D/PhaserEditor/issues/117)

We are putting all our energies on move forward the v2 of the Phaser Editor, but many of you (including clients) are using Phaser v2 and the Phaser Editor v1 series. If there is not a blocking issue, we deliver an update from time to time, with the latest bug fixes.

So keep in contact and send us the issues you find.

Cheers,
