Hi!

A new version of Phaser Editor 2D v3 is ready! It includes some bug fixes and the latest Phaser: v3.24.1.

This is a minor release to keep the editor in synchronization with Phaser.

## Games and tutorials

If you are creating games, tutorials, or any other content (no matter if it is your first project) with Phaser Editor, please, tell us, we will be proud to feature it in our website, newsletter, and social channels.

## Collaboration

A lot is happening in our [Discord server](https://discord.com/invite/4DdpMMD). Join us for collaboration. Ask questions or tell us your ideas. Your feedback is highly appreciated and is key in the process of providing a more friendly and stable IDE.

We thank [Gravix](https://twitter.com/ItsGravix/) for his collaboration with the administration of the server. He is doing an amazing job!

## Bug fixes

* [#52](https://github.com/PhaserEditor2D/PhaserEditor2D-v3/issues/52) Moving objects with arrow keys is not updating the properties in the Inspector view.
* [#54](https://github.com/PhaserEditor2D/PhaserEditor2D-v3/issues/54) Asset Pack Editor: fixes delete items with context menu Delete option.
* Scene Editor: fixes Asset Pack file selection dialog in **Compiler Scene Settings**.
* Editors clear undo history when a file is modified externally.

## Phaser v2.24.1

The main reason of this release of the editor is to get the latest changes from Phaser.

[Read the changelog of Phaser 3.24.1](https://github.com/photonstorm/phaser/blob/master/CHANGELOG.md#version-3240---rem---13th-july-2020)

## What's next

We keep working on the new User Component feature. Maybe in two weeks, it will be ready. We are excited about it. We developed a demo to show how you can use it in real scenarios. Maybe we will make a video too. User components can be used for different purposes, but what we think has more value is that you can split coding from scene making. In theory, a Phaser skilled dev can code a set of components that another member of the team can use to build the game levels without the need to touch the code.

Maybe, someday,  could be realized the idea of a "prefab and components framework" to create a particular class of games. Or create reusable components and publish them in a marketplace.

Tomorrow we should release v2 of the editor, with the latest Phaser.
